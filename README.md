
# Week 1 JavaScript Practice

This repository contains beginner-to-intermediate JavaScript practice files organized by topic.

## Folder Structure

- `array-operations/` – exercises for working with arrays and array methods
- `array-operations-adv/` – more advanced array-related assignments
- `collections/` – examples related to collections and data types
- `control statements/` – conditionals, loops, and related assignments

## How to Use

1. Open the folder that matches the topic you want to practice.
2. Run the JavaScript file using Node.js.
3. Review the code and modify it to experiment with different inputs or logic.

## Example

```bash
node array-operations/marks.js.js
```

## Notes

These files are intended for learning and practice. Feel free to add more examples or improve the code as you continue learning JavaScript.
